# Getting Started

- FullStack TodoList App

- Next.js
- TailwindCSS
- ShadCN-UI（zod ReactHookForm）
- Zustand
- Tanstack Query
- Prisma
- pgSQL(Neon)
- FrondEnd Stack(prettier eslint-airbnb husky commitizen cz lint-staged tsconfig.json)

First, run the development server:

    ```bash
    #husky
    bun prepare
    #setup husky
    bun setup-husky


    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
    ```
